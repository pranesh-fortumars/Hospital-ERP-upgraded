# 🏥 Antigravity Health - Enterprise Healthcare ERP

An enterprise-grade, production-ready Healthcare ERP and Telemedicine platform designed for modern hospitals, multi-specialty clinics, and healthcare networks. Built with **Next.js 16**, **Tailwind CSS v4**, and **Lucide Icons**.

## 🌟 Key Features

### 👨‍⚕️ Clinical & Paramedical Modules
- **Patient 360 & RPM**: Complete patient clinical history with real-time Remote Patient Monitoring (RPM) for vitals tracking.
- **HD Telemedicine**: Secure video and audio consultation session with an integrated **AI Clinical Insight Engine** for live SOAP note generation.
- **AI Symptom Checker**: Patient-centric assistant for preliminary assessment and automated specialist routing.
- **Diagnostic Lab (LIS)**: Real-time sample tracking and laboratory machine status monitoring.
- **E-Pharmacy**: Comprehensive pharmacy inventory management with automated reorder intelligence.

### 🏢 Operational & Administrative Modules
- **Hospital Operations Control**: Real-time bed inventory management, ward occupancy tracking, and staff attendance monitoring.
- **OT & ICU Command Center**: Live remote monitoring of Operation Theaters and Critical Care units, including environmental and anesthesia data.
- **Enterprise Analytics**: Advanced financial and operational intelligence dashboard for hospital administrators.
- **Security & Compliance Audit**: HIPAA-compliant global audit trail and security incident investigation portal.

### 🎨 Design System
- **Premium Medical Aesthetic**: A sleek Teal/Emerald theme built with **OKLCH** color space for maximum visual fidelity.
- **Glassmorphism UI**: High-end dashboard interface with backdrop blurs, subtle micro-animations, and balanced typography.
- **Dark Mode Optimized**: Fully responsive dark mode with tailored background mesh effects for specialized hospital departments.

## 🚀 Tech Stack
- **Framework**: Next.js 16 (App Router + Turbopack)
- **Styling**: Tailwind CSS v4 (Pure CSS-first architecture)
- **Icons**: Lucide React
- **Charts**: Recharts (Medical Telemetry)
- **State**: React Hooks (Zustand/TanStack Query ready)

## 🛠 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd Hospital_ERP-master
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
